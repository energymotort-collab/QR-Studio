ข้อกำหนดและเงื่อนไขการใช้ QR‑Ixonity

อัปเดตล่าสุด: 11 ตุลาคม 2025, 20:09 EEST • เวอร์ชัน: 2025‑10‑12‑1

ข้อกำหนดและเงื่อนไขนี้ (ต่อไปนี้ — «ข้อกำหนด») ควบคุมการโต้ตอบของคุณกับแอปพลิเคชันมือถือ QR‑Ixonity (แบรนด์ Ixonity) บริการเว็บ โครงสร้างพื้นฐาน backend และฟังก์ชันที่เกี่ยวข้อง (ต่อไปนี้ — «บริการ») ที่ให้โดย Private Entrepreneur Zaitsev Oleksii Yuriyovych (ต่อไปนี้ — «เรา», «ของเรา», «ผู้ควบคุม») การใช้บริการแสดงว่าคุณยอมรับข้อกำหนดเหล่านี้ หากไม่ยอมรับ — หยุดใช้บริการ เราเพียรพยายามทำให้กฎเหล่านี้ชัดเจนและเป็นธรรมที่สุด

1. ข้อกำหนดทั่วไป

1.1 การยอมรับข้อกำหนด

การใช้บริการหมายถึงการยอมรับข้อกำหนดนี้และนโยบายความเป็นส่วนตัวของเราเต็มรูปแบบ (มีในแอป) เอกสารทั้งสองมีผลทางกฎหมาย เราสงวนสิทธิ์ในการอัปเดตข้อกำหนด จะแจ้งการเปลี่ยนแปลงที่สำคัญในแอปพร้อมวันที่ใหม่ การใช้งานต่อเนื่องหลังการเปลี่ยนแปลงหมายถึงการยอมรับของคุณ

1.2 วัตถุประสงค์ของบริการ

QR‑Ixonity ใช้สำหรับสร้าง สไตล์ และวิเคราะห์ QR Code เพื่อวัตถุประสงค์ส่วนตัวหรือเชิงพาณิชย์ เราไม่รับผิดชอบวิธีการใช้ QR Code ที่สร้าง หากละเมิดกฎหมาย

1.3 ภาษาและเขตอำนาจศาล

ข้อกำหนดนี้มีในภาษาของสหภาพยุโรป และภาษาราชการของสหราชอาณาจักร สหรัฐ แคนาดา และประเทศอื่นๆ ในกรณีขัดแย้ง เวอร์ชันในภาษาแม่ของคุณหรือภาษาอังกฤษมีผลเหนือกว่า เว้นแต่ระบุเป็นอย่างอื่น เขตอำนาจศาลขึ้นอยู่กับถิ่นที่อยู่ของคุณ: ยูเครน (กฎหมายยูเครน «ว่าด้วยการคุ้มครองสิทธิผู้บริโภค») สหภาพยุโรป (GDPR) สหราชอาณาจักร (UK GDPR, Data Protection Act 2018) สหรัฐ (Federal Trade Commission Act, CCPA/CPRA) แคนาดา (PIPEDA) และกฎหมายที่เกี่ยวข้องของประเทศอื่นที่คุณใช้บริการ

1.4 การยอมรับในแอปและการเก็บความยินยอม (ล็อกความยินยอม)

เมื่อลงทะเบียนในแอป แสดงลิงก์ไปยังเวอร์ชันปัจจุบันของข้อกำหนดนี้และนโยบายความเป็นส่วนตัว คุณสามารถดูเอกสารโดยตรงในแอป จากนั้นยืนยันความยินยอมด้วยการกระทำที่ชัดเจน (ช่องทำเครื่องหมายหรือปุ่ม) เก็บบันทึกการยอมรับ (ประเภทเอกสาร — ข้อกำหนด/นโยบาย เวอร์ชัน/วันที่ ท้องถิ่น เวลายืนยัน) ในระบบเซิร์ฟเวอร์ของเรา (Firebase/Firestore) และในแอปแบบท้องถิ่นเฉพาะเพื่อการตรวจสอบ เพื่อยืนยันความถูกต้องตามกฎหมายของการประมวลผลและปฏิบัติตามข้อกำหนดทางกฎหมาย

หากทำการเปลี่ยนแปลงที่สำคัญในข้อกำหนดนี้ จะแสดงเวอร์ชันที่อัปเดตในแอปอีกครั้งและร้องขอการยืนยันความยินยอม หากไม่ยอมรับการอัปเดต หยุดใช้บริการและ/หรือลบบัญชี

ระยะเวลาการเก็บล็อกความยินยอม: เก็บล็อกการยอมรับในระหว่างการมีอยู่ของบัญชีของคุณและสูงสุด 3 ปีหลังการลบเฉพาะเพื่อปฏิบัติตามข้อกำหนดทางกฎหมาย การตรวจสอบ และยืนยันความยินยอมที่ให้; หลังระยะเวลานี้ ล็อกถูกลบหรือไม่ระบุตัวตน พื้นฐานทางกฎหมายคือการปฏิบัติตามสัญญาและ/หรือผลประโยชน์ที่ชอบด้วยกฎหมายของเราในการยืนยันความถูกต้องตามกฎหมายของการประมวลผล คุณสามารถคัดค้านการประมวลผลดังกล่าวโดยติดต่อเราผ่านช่องทางที่ระบุในเอกสาร

2. การลงทะเบียนและบัญชี

2.1 การสร้างบัญชี

การเข้าถึงฟังก์ชันเต็มรูปแบบ (การเก็บ QR Code การวิเคราะห์ การซิงค์) ต้องการบัญชีผ่าน Firebase Auth (ใช้ email หรือการล็อกอิน Apple ID; โหมดไม่ระบุชื่อพร้อมข้อจำกัดเป็นไปได้หากจำเป็น) คุณรับผิดชอบความถูกต้องของข้อมูลที่ให้และความปลอดภัยของข้อมูลรับรองตัวตน

2.2 การใช้แบบไม่ระบุชื่อ

คุณสามารถใช้บริการโดยไม่ลงทะเบียน แต่จำกัดการเข้าถึงการเก็บข้อมูล การซิงค์ และการวิเคราะห์ ข้อมูลไม่ระบุชื่อถูกลบเมื่อสิ้นสุดเซสชัน หากไม่เชื่อมโยงกับบัญชี

2.3 หน้าที่ของผู้ใช้

  • อย่าแบ่งปันการเข้าถึงของคุณกับผู้อื่น
  • แจ้งเราทันทีเกี่ยวกับกิจกรรมที่น่าสงสัย (อีเมล: ixonity@gmail.com)
  • อย่าใช้สคริปต์อัตโนมัติในการสร้างบัญชีจำนวนมาก

2.4 การลบบัญชี

คุณสามารถลบบัญชีผ่านการตั้งค่า (/me/delete) นำไปสู่การลบที่ไม่สามารถย้อนกลับได้ของข้อมูลทั้งหมดของคุณ รวมถึง QR Code ล็อกสถิติที่เชื่อมโยง ไฟล์ใน Storage ล็อกการใช้งาน และโทเค็นอุปกรณ์ แนะนำให้ส่งออกข้อมูลสำคัญก่อนลบ

2.5 การวิเคราะห์ QR Code (ตัวเลือก ในแผนที่เสียค่าใช้จ่าย)

การวิเคราะห์ QR Code (เช่น: สถิติการสแกน) เป็นฟังก์ชันตัวเลือก มีในสมัครสมาชิกที่เสียค่าใช้จ่าย — Pro, Business, Ultra เปิดใช้งานเฉพาะด้วยความยินยอมที่ชัดเจนของคุณ (opt‑in) ผ่านสวิตช์ «เปิดสถิติการสแกน» ในตั้งค่าของ QR Code เฉพาะ สำหรับผู้ใช้ใน EU/UK/EEA การวิเคราะห์ทำเฉพาะบนพื้นฐานความยินยอมของคุณในฐานะเจ้าของ QR Code ข้อมูลสแกนเนอร์ไม่ระบุตัวตนและรวม; IP ใช้ครั้งเดียวสำหรับการระบุประเทศ/เมืองและไม่ถูกเก็บ คุณสามารถหยุดการรวบรวมข้อมูลโดยปิดสำหรับ QR Code หรือลบ QR Code เฉพาะ ซึ่งลบสถิติที่เชื่อมโยงทั้งหมดอัตโนมัติจาก backend ของเรา (Firebase) ไม่ใช้ IDFA และไม่ทำ «การติดตาม» ในความหมายของ ATT Apple

3. ใบอนุญาตและสิทธิ์การเป็นเจ้าของ

3.1 ใบอนุญาตสำหรับผู้ใช้

มอบใบอนุญาตที่ไม่ผูกขาด สามารถเพิกถอนได้สำหรับการใช้บริการตามข้อกำหนดนี้ คุณสามารถสร้าง QR Code เพื่อใช้ส่วนตัวหรือเชิงพาณิชย์ แต่ไม่มีสิทธิ์ขายหรือแก้ไขแอปเอง

3.2 เนื้อหาของคุณ

คุณคงสิทธิ์ใน QR Code ที่สร้าง การออกแบบ (โลโก้ กรอบ) และลิงก์ที่เชื่อมโยง ไม่อ้างสิทธิ์ทรัพย์สินทางปัญญาในเนื้อหาของคุณ แต่ได้รับสิทธิ์ในการประมวลผลเพื่อให้บริการ (เก็บในเซิร์ฟเวอร์ การส่ง ฯลฯ)

3.3 สิทธิของเรา

สิทธิทั้งหมดในซอฟต์แวร์ การออกแบบอินเทอร์เฟซ และโลโก้ QR‑Ixonity เป็นของเรา คุณไม่สามารถคัดลอก แจกจ่าย หรือวิศวกรรมย้อนกลับโค้ดโดยไม่ได้รับอนุญาตเป็นลายลักษณ์อักษร

4. การใช้บริการ

4.1 การกระทำที่อนุญาต

  • การสร้าง การปรับแต่ง และการแบ่งปัน QR Code
  • การเปิดการวิเคราะห์เพื่อติดตามการสแกน (หากคุณมีสิทธิ์เข้าถึงที่เหมาะสม)
  • การใช้บริการเพื่อวัตถุประสงค์ที่ถูกกฎหมาย รวมถึงการตลาด การศึกษา หรือโครงการส่วนตัว

4.2 การกระทำที่ห้าม

คุณไม่สามารถ:

  • ใช้บริการเพื่อเผยแพร่เนื้อหาที่ผิดกฎหมาย (ความรุนแรง การฉ้อโกง phishing การละเมิดลิขสิทธิ์)
  • โอเวอร์โหลดเซิร์ฟเวอร์ด้วยคำขอจำนวนมากหรือบอท
  • ทำลายการทำงานของบริการ (โดยเฉพาะผ่านมัลแวร์)
  • ใช้ QR Code เพื่อหลอกลวงหรือฉ้อโกงผู้ใช้
  • สร้าง QR Code ที่นำไปสู่เนื้อหาที่ละเมิดสิทธิมนุษยชน ทรัพย์สินทางปัญญา ความเป็นส่วนตัว หรือสิทธิทางกฎหมายอื่นๆ รวมถึงแต่ไม่จำกัดที่ การเลือกปฏิบัติ ความเกลียดชัง การแสวงประโยชน์ การหมิ่นประมาท หรือการเผยแพร่ข้อมูลเท็จที่ก่อให้เกิดความเสียหาย

4.3 การตรวจสอบ

สงวนสิทธิ์ในการ:

  • บล็อกบัญชีหรือลบเนื้อหาที่ละเมิดข้อกำหนดนี้ ผู้ใช้จะได้รับแจ้งการบล็อกหรือการล้าอีเมลหรือในแอป และมีสิทธิ์อุทธรณ์การตัดสินใจโดยติดต่อ ixonity@gmail.com ภายใน 14 วัน
  • ความร่วมมือกับหน่วยงานบังคับใช้กฎหมายเมื่อมีคำร้องขอที่ถูกกฎหมาย
  • จำกัดการเข้าถึงชั่วคราวระหว่างการอัปเดตทางเทคนิค (แจ้งการอัปเดตที่สำคัญผ่านการแจ้งเตือน push หรืออีเมลหากส่งผลต่อการเข้าถึงบริการ)

5. ข้อจำกัดความรับผิด

5.1 ข้อจำกัดทางเทคนิค

บริการให้ «ตามที่มี» ไม่รับประกันการทำงานต่อเนื่องหรือไม่มีข้อผิดพลาดเนื่องจากความพึ่งพาแพลตฟอร์มบุคคลที่สาม (Google Firebase, Apple) ในกรณีความล้มเหลว จะพยายามแก้ไข

5.2 ความเสียหายทางอ้อม

ไม่รับผิดชอบต่อ:

  • การสูญหายของข้อมูลเนื่องจากการป้อนที่ไม่ถูกต้องหรือการลบบัญชีโดยผู้ใช้
  • ความเสียหายจากใช้ QR Code (เช่น: หากลิงก์นำไปยังเว็บไซต์อันตราย)
  • การตีความการวิเคราะห์ที่ไม่ถูกต้องเนื่องจากลักษณะทางเทคนิค

5.3 ความรับผิดสูงสุด

ความรับผิดของเราจำกัดที่จำนวนเงินที่คุณใช้จ่ายในบริการ (หากมี) และไม่เกิน 1000 hryvnia (หรือเทียบเท่าในสกุลเงินของคุณ) ข้อจำกัดนี้ไม่ใช้ในที่ที่ห้ามหรือจำกัดโดยกฎหมายคุ้มครองผู้บริโภคที่บังคับในเขตอำนาจศาลของคุณ โดยเฉพาะในกรณีการกระทำโดยจงใจหรือความประมาทเลินเล่อร้ายแรง

8.1 การแจ้งเตือน Push

  • ใช้การอนุญาตระบบ iOS/iPadOS สำหรับการแจ้งเตือน หากให้ สามารถส่งการแจ้งเตือนที่จำเป็นทางเทคนิค (เช่น: เกี่ยวกับขั้นตอนการสแกนที่สำคัญ ข้อความบริการเกี่ยวกับขีดจำกัด/สถานะการสมัครสมาชิก)
  • คุณสามารถเปลี่ยนการอนุญาตได้ทุกเมื่อในตั้งค่าอุปกรณ์ (Settings → Notifications) หรือปิดเกณฑ์ขั้นตอนเดี่ยวในส่วน «การแจ้งเตือน» ของแอป
  • การแจ้งเตือน push ทางการตลาดไม่ส่งในปัจจุบัน หากมีตัวเลือกดังกล่าวในอนาคต จะมีเฉพาะหลังความยินยอมที่ชัดเจนแยกต่างหาก (opt‑in) ในแอป; สามารถยกเลิกได้ทุกเมื่อ
  • โทเค็นอุปกรณ์ใช้เฉพาะสำหรับการส่งการแจ้งเตือนผ่าน Apple/FCM และไม่สำหรับการติดตามโฆษณาหรือสร้างโปรไฟล์

10. ข้อกำหนดสุดท้าย

ข้อกำหนดนี้เป็นข้อตกลงฉบับสมบูรณ์ระหว่างคุณและเราเกี่ยวกับการใช้บริการและแทนที่ข้อตกลงก่อนหน้า หากข้อกำหนดใดถูกตัดสินว่าเป็นโมฆะ ข้อกำหนดอื่นยังคงมีผล การใช้บริการต่อเนื่องหลังการอัปเดตข้อกำหนดหมายถึงการยอมรับ

11. แผนและการสมัครสมาชิกที่เสียค่าใช้จ่าย

บริการมีแพ็กเกจฟรีพื้นฐาน (Start) และการสมัครสมาชิกที่เสียค่าใช้จ่ายผ่าน Apple StoreKit เพื่อเข้าถึงฟังก์ชันขั้นสูง (รวมการวิเคราะห์ QR Code, QR Code แบบไดนามิก และ API): Pro, Business, Ultra

  • รายละเอียดการสมัครสมาชิก (รายการฟังก์ชัน ระยะเวลา ราคา) ระบุในแอปและหน้า App Store การสมัครสมาชิกต่ออายุอัตโนมัติ เว้นแต่ยกเลิก 24 ชั่วโมงก่อนสิ้นงวด
  • การยกเลิก: เป็นไปได้ผ่านการตั้งค่า Apple ID ตามกฎ Apple StoreKit
  • การคืนเงิน: ควบคุมโดยนโยบาย Apple ติดต่อฝ่ายสนับสนุน Apple สำหรับคำร้องคืนเงิน ไม่เก็บข้อมูลการชำระเงินและไม่ควบคุมการประมวลผลการชำระเงินหรือการคืนเงิน
  • ข้อจำกัดการเข้าถึง: หลังการสมัครสมาชิกที่เสียค่าใช้จ่ายหมดอายุ การเข้าถึงฟังก์ชันที่เสียค่าใช้จ่ายจะหยุด แต่ QR Code ที่สร้างยังคงใช้งานได้ หากคุณไม่ลบ

12. การปฏิบัติตามข้อกำหนด Apple

  • App Tracking Transparency (ATT): ไม่ใช้ IDFA หรือเทคโนโลยีการติดตามโฆษณาอื่นๆ การวิเคราะห์ QR Code ไม่ระบุตัวตนและเปิดใช้งานเฉพาะด้วยความยินยอมของคุณ (ดูส่วน 2.5)
  • StoreKit: การชำระเงินทั้งหมดประมวลผลผ่าน Apple StoreKit เท่านั้น เพื่อให้สอดคล้องกับกฎ Apple

13. การแบ่งปันข้อมูล

ข้อมูลบางส่วนที่รวบรวมผ่านบริการ (เช่น: ผ่าน Firebase) อาจถ่ายโอนไปยังสหรัฐ การถ่ายโอนดังกล่าวได้รับการคุ้มครองโดยข้อตกลงมาตรฐาน (SCC) หรือ Data Privacy Framework (DPF) ตาม GDPR/UK GDPR ตามที่อธิบายในนโยบายความเป็นส่วนตัว ไม่ขายข้อมูลของคุณและไม่ใช้สำหรับโฆษณาแบบกำหนดเป้าหมาย

14. ข้อจำกัดอายุ

บริการตั้งใจสำหรับผู้ใช้ตั้งแต่อายุ 17 ปี (หรืออายุความยินยอมในเขตอำนาจศาลของคุณ) หากพบว่าบัญชีเป็นของบุคคลที่อายุต่ำกว่านี้ จะถูกลบตามกฎ COPPA (สหรัฐ) และ GDPR (มาตรา 8)

15. เหตุสุดวิสัย

ไม่รับผิดชอบต่อการไม่ปฏิบัติหรือความล่าช้าในการให้บริการเนื่องจากเหตุสุดวิสัย รวมถึงแต่ไม่จำกัดที่ ภัยพิบัติธรรมชาติ สงคราม การนัดหยุดงาน การโจมตีทางไซเบอร์ หรือความล้มเหลวในการทำงานของแพลตฟอร์มบุคคลที่สาม (เช่น: Google Firebase, Apple) จะแจ้งคุณเกี่ยวกับสถานการณ์ดังกล่าวผ่านการแจ้งเตือน push หรืออีเมลและพยายามกู้คืนการทำงานของบริการ
