2.4 Suppression de Compte

       

Vous pouvez supprimer votre compte via les paramètres (/me/delete). Cela entraînera la suppression irréversible de toutes vos données, y compris les codes QR, les enregistrements statistiques associés, les fichiers dans Storage, les enregistrements d'utilisation (usage‑записи) et les jetons d'appareil. Avant de supprimer, nous vous recommandons d'exporter les informations importantes.

       

Veuillez noter: la suppression du compte dans l'application n'entraîne pas l'annulation automatique d'un abonnement actif. Vous pouvez annuler le renouvellement automatique et gérer votre abonnement dans les paramètres Apple ID (Paramètres → Votre nom → Abonnements).

       

2.5 Analyse des Codes QR (optionnel, dans les forfaits payants)

       

L'analyse des codes QR (par exemple, les statistiques de scan) est une fonctionnalité optionnelle, disponible dans les abonnements payants — Business, Pro, Ultra. Elle est activée uniquement avec votre consentement explicite (opt‑in) via le commutateur «Activer les statistiques de scan» dans les paramètres du code QR spécifique. Pour les utilisateurs de l'UE/UK/EEE, l'analyse est effectuée uniquement sur la base de votre consentement en tant que propriétaire du code QR. Les données des scanneurs sont anonymisées et agrégées; l'adresse IP est utilisée une seule fois pour déterminer le pays/la ville et n'est pas stockée. Vous pouvez arrêter la collecte d'analyses en la désactivant pour le code QR ou en supprimant le code QR spécifique, ce qui supprime automatiquement toutes les données statistiques associées de notre back-end (Firebase). Nous n'utilisons pas d'IDFA et n'effectuons pas de «tracking» au sens d'Apple ATT.
